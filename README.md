
📊 Bank Churn Analysis Dashboard
📌 Overview
This project presents a Bank Customer Churn Analysis Dashboard built using Power BI. It analyzes customer behavior, identifies churn patterns, and provides actionable insights to improve customer retention and business decision-making.
🎯 Objectives
Analyze customer churn trends
Identify factors influencing customer exit
Compare active vs inactive customers
Provide insights to improve retention strategies
🛠️ Tools Used
Power BI
Excel
DAX
📂 Dataset
The dataset includes customer details such as:
Customer activity (Active / Inactive)
Credit card ownership
Credit score categories
Gender
Churn status (Exited / Retained)
Year and Month
📊 Key Metrics
Total Customers: 10,000
Active Customers: 5,151
Inactive Customers: 4,849
Exited Customers: 2,037
Retained Customers: 7,963
Credit Card Holders: 7,055
Non-Credit Card Holders: 2,945
📈 Insights
Inactive customers have a higher churn rate
Customers with lower credit scores are more likely to exit
Retention is significantly higher than churn
Credit card holders show better retention behavior
📊 Dashboard Features
Churn analysis by credit score
Active vs inactive customers (year-wise)
Gender-based churn distribution
Interactive filters (Year & Month)
🚀 How to Use
Download the .pbix file
Open in Power BI Desktop
Use filters to explore insights
Interact with visuals for detailed analysis
